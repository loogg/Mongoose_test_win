# 术语定义

## 示教器

本设备，作为主控端，提供 Web 控制台界面。

| 信息 | 说明 |
|------|------|
| 固件版本 | 示教器软件版本 |
| 硬件版本 | 示教器硬件版本 |
| 名称 | 设备名称 |
| 序列号 | 设备唯一标识 |

---

## 工具

通过 RS485 总线连接的外部设备。

### 连接特性
- 总线上可挂载多把工具
- 同时只连接（激活）一把

### 工具状态
| 状态 | 说明 |
|------|------|
| 离线 | 没有设备被激活 |
| 连接中 | 正在建立连接，示教器需要通信成功后才会把状态变为在线，如果在线期间通信断开，状态会变为连接中 |
| 在线 | 已连接，可正常通信 |

### 工具信息来源
| 信息 | 来源 |
|------|------|
| 名称 | 存储在示教器（用户配置） |
| 固件版本 | 从工具读取（在线时） |
| 硬件版本 | 从工具读取（在线时） |
| 型号 | 从工具读取（在线时） |
| 序列号 | 从工具读取（在线时） |

## 网路服务

示教器支持2路自定义客户端接入和3路Modbus TCP 客户端接入
